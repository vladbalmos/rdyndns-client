# rdyndns-client

## Requirements
- GNU/Linux
- [bundler.io](http://bundler.io/)
- [ipaddress](https://github.com/ipaddress-gem/ipaddress) >= 0.8.3
- [nsupdate](http://linux.die.net/man/8/nsupdate)

## Installation
```bash
git clone https://github.com/vladbalmos/rdyndns-client
cd rdyndns-client
bundle install
```
